# Stack
- This is a datastructure that uses the last in first out principle(LIFO)

## Operations in a stack
- [x] pop() used to remove the last element in a stack
- [x] append used to add elements to a stack

## Importance of stacks
- backtracking, used to find the correct path through a maze
- compile time memory managemen, nested and recursive funtions
- depth first search
